# windasekardewi.github.io

Personal portfolio site for Winda Sekar Dewi — Senior Technical Program Manager.

Live at [windasekardewi.github.io](https://windasekardewi.github.io)

## Stack

Plain HTML + CSS. No framework, no build step, no dependencies.

## Structure

```
index.html    main portfolio page
style.css     all styles
```

## To update

Edit `index.html` or `style.css` directly, then commit and push.
GitHub Pages deploys automatically within ~2 minutes.
